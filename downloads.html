⚙ Connecting to the Network

The wallet finds peers automatically on first launch. If it stays at 0 connections, add this to hnb.conf:

addnode=23.88.122.228:28888

server=1

listen=1

On Windows the config lives at %APPDATA%\HookersAndBlow\hnb.conf

On Linux: ~/.hnb/hnb.conf
